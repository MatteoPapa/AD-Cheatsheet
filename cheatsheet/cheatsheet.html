Setup

Wireguard Setup

Attiva una configurazione WireGuard specificata nel file di configurazione (FILECONF).
sudo wg-quick up FILECONF

Wireguard Status

Mostra lo stato attuale dell'interfaccia WireGuard, comprese le chiavi, peer e traffico.
wg show

Attack

Eseguire exploit contro NOP Team

Esegue lo script di exploit contro il NOP Team
python3 exploit.py 10.60.0.1

Linkare exploit con TOTTI

Esegue periodicamente lo script di exploit contro tutti i team, e manda i risultati a TOTTI
python3 start_sploit.py -u http://10.80.1.2:5000 exploit.py

SSH

SSH nella Macchina del Team

Accede alla macchina del team tramite SSH
ssh root@10.60.1.1

Zippare tutti i file nella directory corrente

Crea un archivio .zip di tutti i file e cartelle nella cartella attuale
zip -r zipName.zip .

Scarica File dalla Macchina del Team

Copia un file dalla macchina remota alla macchina locale
scp root@10.60.1.1:/root/zipName.zip .

Docker

Elenco Container Attivi

Mostra tutti i container Docker attualmente in esecuzione
docker ps

Buildare un servizio Docker (senza cache)

Prebuilda i container da zero ignorando la cache
docker compose build --no-cache

Riavvia i Servizi Docker

Ferma i container e li riavvia usando immagini giÃ  buildate (detached)
docker compose down && docker compose up -d

Accesso al Container Docker

Apre una shell interattiva nel container Docker specificato
docker exec -it DOCKER_ID bash

Visualizza Log di Docker

Mostra i log di un servizio Docker
docker compose logs
